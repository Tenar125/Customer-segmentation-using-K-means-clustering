# Customer-segmentation-using-K-means-clustering


## Project Overview

This project focuses on segmenting mall customers based on their purchasing behavior using **Unsupervised Machine Learning (K-Means Clustering)**. The goal is to identify hidden customer groups and help businesses design targeted marketing strategies.

##  Objective

To group customers into meaningful segments based on **income and spending behavior** and develop data-driven marketing strategies for each segment to improve customer engagement and business growth.

##  Dataset

**Mall Customers Dataset**

Features used:
- Age
- 
- Gender
- 
- Annual Income
- 
- Spending Score

##  Key Techniques Used

- Exploratory Data Analysis (EDA)
- 
- K-Means Clustering
- 
- Elbow Method (to find optimal clusters)
- 
- PCA / t-SNE (for visualization)
- 
- Data visualization using charts


## Data Quality Observations

- No missing values were found in the dataset. If present, appropriate imputation techniques could be applied.
- No duplicate records were detected, ensuring data accuracy and reliability.

## Data Insights

- Customers have a wide income range (15–137) and spending scores up to 99, showing both low and high-value customers.
- 
- Majority of customers belong to **adults and seniors**, indicating these are key target groups.
- 
- Seniors and adults show higher average income (60–65), making them more valuable for targeting.
- 
- Male customers have higher average income, while female customers show higher spending scores.
- 
- Age-level missing values were removed before visualization for accurate analysis.
  
## Customer Segments & Insights

 **Average Customers (77)**  
  Majority segment with medium income and medium spending behavior.

-  **VIP Customers (High Income & High Spending - 39)**  
  High-value customers. Should be rewarded with VIP benefits and loyalty programs.

- **High Income, Low Spending Customers**  
  Have purchasing power but low engagement. Need targeted ads and personalized offers.

- **Low Income, Low Spending Customers**  
  Should be attracted using discounts and budget-friendly marketing strategies.

- **Low Income, High Spending Customers**  
  Loyal buyers despite low income. Should be retained with loyalty rewards and special deals.

##  Conclusion
This project demonstrates how machine learning can be used to understand customer behavior and create effective marketing strategies that improve business performance and customer satisfaction.
